# Dashboard 메뉴 설계

**날짜:** 2026-05-11  
**상태:** 확정  
**스코프:** 3탭 구조 (투자 탭 스펙아웃)

---

## 목표

가계부 앱에 **DASHBOARD** 메뉴를 추가한다. 지출/수입/자산의 큰 흐름과 분석을 한 화면에서 파악할 수 있도록 한다.

- 연간 추세 중심, 기간 선택 가능
- 순자산과 투자 포트폴리오 요약이 핵심 지표
- 탭별로 영역을 분리해 깊은 분석 제공

---

## 구조

### 탭 구성

```
Dashboard
├── 개요 탭  (기본 진입 탭)
├── 지출 탭
└── 자산 탭
```

투자 탭은 현재 스코프에서 제외. 투자 요약은 개요 탭에 포함.

### 기간 선택기 (전역)

- 위치: 개요 탭 상단 우측
- 선택지: 이번달 / 이번 분기 / 올해 / 커스텀(날짜 범위 직접 입력)
- 동작: 선택 시 모든 탭의 차트·수치에 동일하게 적용
- 예외: 자산 탭의 자산 구성 도넛 및 항목 리스트는 항상 최신 기준

---

## 탭별 설계

### 1. 개요 탭

스크롤 순서 (위 → 아래):

| # | 컴포넌트 | 내용 |
|---|---------|------|
| ① | 기간 선택기 | 전역 기간 조정 |
| ② | 순자산 히어로 카드 | 총자산 - 부채 = 순자산, 전월/전년 대비 변화금액 + 방향 |
| ③ | 수지 요약 + 투자 요약 (2컬럼) | 수입/지출/저축액 \| 원금/평가금/수익률 |
| ④ | 순자산 변화 바 차트 | 기간 내 월별 순자산 추이 (12개월) |
| ⑤ | 지출 TOP 5 카테고리 | 수평 미니 바 차트 + 금액 |
| ⑥ | 최근 거래 3~5건 | 지출/수입/투자 최신 순, "더보기" 탭 시 해당 탭으로 이동 |

---

### 2. 지출 탭

스크롤 순서:

| # | 컴포넌트 | 내용 |
|---|---------|------|
| ① | 총 지출 헤더 카드 | 기간 총 지출, 전월 대비 증감, 기간 내 월평균 |
| ② | 카테고리별 도넛 차트 | 지출 카테고리 비중, 범례 탭 시 카테고리 상세 이동 |
| ③ | 월별 지출 바 차트 | 기간 내 월별 지출 추이, 이번달 강조 |
| ④ | 카테고리 상세 리스트 | 금액 + 비중 바 + 전월 대비 ▲▼, 탭 시 거래 목록 이동 |
| ⑤ | 최대 단건 지출 TOP 5 | 가장 큰 단건 지출 항목 |

---

### 3. 자산 탭

스크롤 순서:

| # | 컴포넌트 | 내용 |
|---|---------|------|
| ① | 순자산 헤더 | 총자산 / 부채 / 순자산 3단 표시, 전년 대비 성장액 |
| ② | 자산 구성 도넛 차트 | 현금/예금 · 투자자산 · 부동산 · 기타 비중 + 금액, 항상 최신 기준 |
| ③ | 순자산 장기 라인 차트 | 가입 이후 ~ 현재 순자산 변화, 기간 선택기 연동 |
| ④ | 자산 항목별 리스트 | 개별 자산 항목 + 금액, 카테고리 색상 점으로 구분 |
| ⑤ | 부채 현황 | 부채 항목별 + 총 부채, 부채 없으면 섹션 전체 숨김 |

## 필요한 백엔드 API

기존 CRUD 데이터를 집계하는 API들이 필요하다. 새로운 데이터 모델 추가는 없고, 기존 트랜잭션 데이터를 집계·요약하는 엔드포인트를 추가한다.

| API | 설명 | 비고 |
|-----|------|------|
| `GET /dashboard/summary` | 기간별 수입/지출/저축/투자 요약 | 개요 탭 ③ |
| `GET /dashboard/net-worth/history` | 월별 순자산 이력 | 개요 탭 ④, 자산 탭 ③ |
| `GET /dashboard/expense/category-breakdown` | 기간별 카테고리별 지출 집계 | 지출 탭 ②④ |
| `GET /dashboard/expense/monthly` | 월별 지출 합계 | 지출 탭 ③ |
| `GET /dashboard/expense/top-transactions` | 최대 단건 지출 TOP N | 지출 탭 ⑤ |
| `GET /dashboard/asset/composition` | 현재 자산 구성 (카테고리별 합계) | 자산 탭 ② |
| `GET /transactions/recent` | 최근 거래 N건 | 개요 탭 ⑥ |

모든 집계 API는 `from` / `to` 쿼리 파라미터로 기간 필터링을 지원한다.

## 스코프 아웃

- 투자 탭 (포트폴리오 상세, 자산별 수익률, 투자 추가금 차트)
- 예산 목표 설정 및 달성률
- 푸시 알림 / 지출 경고
- 외부 시세 API 연동 (주식/코인 실시간 가격)
